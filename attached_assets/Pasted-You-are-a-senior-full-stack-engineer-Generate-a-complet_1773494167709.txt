You are a senior full-stack engineer. Generate a **complete production-ready Cloud PHP Lab system** that works like an **Online XAMPP environment inside the browser**. The platform will allow students to code PHP projects, manage MySQL databases, preview websites, and download their project with database backup.

The project must be secure, scalable, and suitable for deployment on a cloud server such as Render or a VPS.

---

## SYSTEM GOAL

Create a cloud development environment where students can:

• log in to their account
• create PHP projects
• edit files using a browser code editor
• run PHP code
• create and manage MySQL databases
• preview websites live
• download project files with database export

The system should behave like a simplified **cloud version of XAMPP**.
